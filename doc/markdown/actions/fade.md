

## Fade Action - #fade {}

  Call JQuery's [fadeOut(speed)](http://docs.jquery.com/Effects/fadeOut) on the target element.


### Usage

```erlang
   wf:wire(myDiv, #fade { speed=500 })

```

### Attributes

   * `speed` (integer) - Speed of the effect, in milliseconds.
